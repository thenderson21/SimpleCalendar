Simple Events Calendar Help

Use this calendar to track events and blackout dates by day. Click any day to view details and edit statuses.

File menu

  • New calendar clears all events and blackouts.
  • New event opens the event editor.
  • New blackout date opens the blackout picker.
  • Open... loads a saved JSON calendar file.
  • Save... exports your calendar as JSON.

Day details

  • Toggle Blackout to block a date.
  • Use vendor/performer toggles and status dropdowns to update entries.
