Simple Events Calendar Help

A lightweight yearly planner for events and blackout dates.

Quick start

  • Use File → New Event to add a new event and pick one or more dates.
  • Use File → New Blackout Date to create a blackout group and select dates.
  • Click any day in the calendar to view and edit its events and blackout groups.

Calendar navigation

  • Use the year selector to move between years.
  • Today is highlighted. Blackout dates show a red ring and a bold X.

Events

Create an event

  • Choose File → New Event.
  • Enter a title, optional notes, and select one or more dates.
  • Set the type (Vendor or Performer) and the status for each.

Edit an event

  • Click a day, then use the edit icon next to the event.
  • Update title, notes, dates, type, or status and save.

Blackout dates

Blackout groups

  • Blackout dates are stored in groups with a title and notes.
  • Create a group from File → New Blackout Date.
  • Select one or more dates in the blackout picker.

Editing blackout dates

  • Click a day and toggle the Blackout switch to add/remove that date.
  • If the date is not part of a group yet, you can pick an existing group or create a new one.
Conflicts: If you schedule an event on a blackout date (or add a blackout date that overlaps an event), the app asks for confirmation.

Menus and file actions

  • File → New Calendar clears all events and blackout dates.
  • File → Open... opens a .sevc calendar file.
  • File → Save... saves the current calendar as a .sevc file.

Import / Export

  • Import and Export JSON are intended for sharing or backup.
  • Exported JSON contains events and blackout groups (and may omit settings depending on the format).

Settings

  • Use the Settings menu to customize Vendor and Performer labels and colors.
  • Settings are stored in the calendar file when using .sevc.

Troubleshooting

  • If Help doesn’t open, rebuild the app to ensure the help bundle is packaged.
  • If a file doesn’t open, confirm it has the .sevc extension.
